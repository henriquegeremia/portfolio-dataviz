Desminados - Gov Colombia + UN

Project Management BI Consulting Data Visualization

Project Overview

Led the development of an embedded analytics dashboard in collaboration with the Colombian Government and United Nations to visualize minefield risks. The solution enhances humanitarian decision-making through data-driven insights, powered by Looker Studio and Python-based ETL processes.

Key Responsibilities

  • Led project management and stakeholder communication with international teams
  • Designed and implemented ETL processes for complex geospatial data
  • Created interactive dashboards focusing on risk visualization and analysis
  • Developed data models to support humanitarian decision-making processes

Technical Implementation

The project utilized Looker Studio for dashboard creation, with Python-based ETL processes handling complex data transformations. The implementation focused on creating intuitive visualizations for risk assessment while ensuring data accuracy and accessibility for humanitarian teams in the field.

Impact

This solution has become an essential tool for humanitarian teams working on minefield risk assessment in Colombia, providing clear, actionable insights that help protect lives and optimize resource allocation in affected areas.
